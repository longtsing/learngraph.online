# 6.0 本章介绍

## 让 Agent 并发执行

现代 AI Agent 需要同时：
- 调用多个 API
- 并行处理任务
- 响应用户输入

异步编程让这一切成为可能。

## 学习目标

- ✅ async/await 基础
- ✅ 异步上下文管理器
- ✅ 并发执行任务
- ✅ 实战：异步 LangGraph Agent

## 为什么异步很重要？

```python
# 同步：慢
result1 = call_api_1()  # 等待 2 秒
result2 = call_api_2()  # 等待 2 秒
# 总共 4 秒

# 异步：快
results = await asyncio.gather(
    call_api_1(),  # 并行
    call_api_2()   # 并行
)
# 总共 2 秒
```
