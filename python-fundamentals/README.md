# AI 时代学 Python

欢迎来到《AI 时代学 Python》！

## 📚 本书简介

本书是一本专为 AI 时代设计的 Python 学习教程。零基础入门，学会使用 AI 工具辅助编程，通过实战项目快速上手 Python 编程。

## 🎯 学习目标

- 掌握 Python 基础语法和核心概念
- 理解面向对象编程思想
- 学会使用 Python 进行数据处理
- 掌握文件操作和网络编程基础
- 学会使用 AI 工具辅助编程（ChatGPT、Copilot 等）
- 了解 Python 在 AI 领域的应用

## 📖 章节目录

### Module 1: Python 入门
Python 简介、安装配置、第一个程序

### Module 2: 变量与数据类型
变量、数字、字符串、布尔值、类型转换

### Module 3: 流程控制
if-else 条件判断、for/while 循环、异常处理

### Module 4: 函数与模块
函数定义、参数传递、模块导入、包管理

### Module 5: 数据结构
列表、元组、字典、集合、推导式

### Module 6: 文件操作
文件读写、CSV/JSON 处理、路径操作

### Module 7: 面向对象编程
类与对象、继承、多态、魔法方法

### Module 8: 包管理与虚拟环境
pip 使用、虚拟环境、requirements.txt

### Module 9: 网络爬虫基础
requests 库、BeautifulSoup、API 调用

### Module 10: AI 辅助编程
ChatGPT 编程助手、GitHub Copilot、AI 调试技巧

## 💡 学习建议

1. **边学边练**：每个概念都要动手实践
2. **善用 AI**：学会用 ChatGPT 解决编程问题
3. **多写代码**：编程能力来自大量练习
4. **理解原理**：不要死记硬背，要理解为什么
