# 7.0 本章介绍

## 连接真实世界

> **🎯 小白理解**：为什么 Agent 需要"连接真实世界"？
>
> 前面几章，我们学的都是 Python 的基础能力。但一个真正有用的 AI Agent 需要：
>
> ```
> 用户提问 → Agent 思考 → 调用外部服务 → 返回结果
>                              ↓
>                    ┌─────────┴─────────┐
>                    │                   │
>                OpenAI API          天气 API
>               （获取 AI 回答）      （获取天气信息）
> ```
>
> **本章你会学到**：
>
> | 技能 | 作用 | 类比 |
> |------|------|------|
> | 环境变量 | 安全存储 API 密钥 | 保险箱存放贵重物品 |
> | HTTP 请求 | 和外部服务通信 | 打电话给别人 |
> | API 调用 | 使用别人提供的功能 | 点外卖（你下单，饭店做饭）|
>
> **为什么这章很重要？**
>
> - 没有 API = Agent 只能自言自语
> - 有了 API = Agent 能调用 ChatGPT、搜索引擎、数据库...
> - 这是从"玩具程序"到"真实应用"的关键一步！

你的 Agent 需要：
- 安全管理 API 密钥（环境变量）
- 调用 LLM API（OpenAI, Anthropic）
- 处理 HTTP 请求（requests, httpx）
- 集成第三方服务

本章将所有知识整合，构建真实可用的 Agent。

## 学习目标

- ✅ 环境变量管理（.env）
- ✅ HTTP 请求（requests）
- ✅ 调用 OpenAI API
- ✅ 调用 Anthropic API
- ✅ 实战：多 API 集成 Agent
