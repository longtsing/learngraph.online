# 7.0 本章介绍

## 连接真实世界

你的 Agent 需要：
- 安全管理 API 密钥（环境变量）
- 调用 LLM API（OpenAI, Anthropic）
- 处理 HTTP 请求（requests, httpx）
- 集成第三方服务

本章将所有知识整合，构建真实可用的 Agent。

## 学习目标

- ✅ 环境变量管理（.env）
- ✅ HTTP 请求（requests）
- ✅ 调用 OpenAI API
- ✅ 调用 Anthropic API
- ✅ 实战：多 API 集成 Agent
